# ThoxContainerization

**This is a fork of [apple/containerization](https://github.com/apple/containerization)** maintained by Thox.ai as the core runtime library for ThoxTainer and **ThoxOS** (ThoxOS Mini, ThoxOS Air, and custom embedded Linux distributions).

## Key Focus Areas for Thox

- Customized minimal Linux kernels for fast-booting secure edge VMs
- vminitd extensions for Thox control plane, model serving, and attestation
- OCI + native ThoxOS image formats
- Strong integration with Apple silicon security features (SEP, virtualization)

See the sibling [ThoxTainer](../ThoxTainer) repository for the user-facing CLI and services.

Upstream contributions policy: General improvements → upstream to Apple. ThoxOS-specific work stays here under Thox.ai stewardship.
